# SN Environment-Parity Audit (2026-06-09)

Static audit of all 49 services for local-dev assumptions that break in the air-gapped prod K8s env, with adversarial verification. Workflow sn-parity-audit (249 agents).

**37 confirmed bugs** (verified severity): 1 critical, 27 high, 8 medium, 1 low. 265 risks flagged total.

Dominant class (33/37): inter-service clients/servers fall back to localhost / 127.0.0.1 when env/Consul-discovery is absent -> connection-refused in K8s. Many are LATENT in the current SN deploy (masked by the _SERVICE_URL/_SERVICE_HOST env already set), but fragile. Durable fix: env-first resolution, K8s-DNS fallback, NEVER localhost.
